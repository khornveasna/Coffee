# ☕ Coffee POS - How to Start

## 🚀 Quick Start (3 Simple Steps)

### 1️⃣ Open Command Prompt
Press `Win + R`, type `cmd`, press Enter

### 2️⃣ Navigate and Start Server
Copy and paste these commands:
```bash
cd "c:\Users\K-VeaSna\Desktop\Coffee POS System"
node server.js
```

### 3️⃣ Open Browser
Go to: **http://localhost:3000**

Login: `admin` / `1234`

---

## 🛑 To Stop

In the Command Prompt window, press: **`Ctrl + C`**

---

## 📋 Quick Commands Reference

| What to Do | Command |
|------------|---------|
| Start server | `node server.js` |
| Stop server | `Ctrl + C` |
| Install dependencies | `npm install` (first time only) |
| Reset database | `node migrate-db.js` |

---

## 🔑 Default Login

- **Username:** `admin`
- **Password:** `1234`

## 💡 Tips

- Keep the Command Prompt window open while using the system
- Server runs on: http://localhost:3000
- First time? Run `npm install` before starting server
- Bookmark http://localhost:3000 for quick access
